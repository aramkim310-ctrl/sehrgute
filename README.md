권장: GitHub Pages로 호스팅 (추천)
- 예상 공개 URL(Pages 활성화 시): https://aramkim310-ctrl.github.io/sehrgute/ch_german.html
- 활성화 방법(간단):
  1. GitHub에서 리포지토리의 Settings 탭으로 이동
  2. 좌측 메뉴에서 "Pages" 선택
  3. "Source"를 `main` 브랜치, `/ (root)`로 선택 후 Save
  4. 몇 분 기다리면 위의 https://aramkim310-ctrl.github.io/sehrgute/ch_german.html 주소로 접속 가능

대체 방법: raw.githubusercontent.com로 직접 열기
- 위의 원시(raw) URL을 브라우저에 붙여넣으면 HTML 파일이 렌더링되어 열립니다. 단, GitHub의 CORS/콘텐츠 정책으로 일부 스크립트 동작(외부 리소스 로드 등)이 제한될 수 있으니, 안정적으로 보려면 GitHub Pages 사용을 권장합니다.
