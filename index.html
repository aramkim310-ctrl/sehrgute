생성형 AI (Generative AI)

간단한 소개 페이지

무엇인가요?

생성형 AI는 텍스트, 이미지, 오디오, 코드 등 새로운 콘텐츠를 자동으로 생성하는 인공지능 기술을 말합니다. 주로 대규모 학습된 모델(예: 언어 모델, 생성적 적대 신경망 등)을 사용합니다.

주요 예시

  • 텍스트 생성: 글쓰기, 요약, 번역, 대화
  • 이미지 생성: 스타일 기반 이미지 생성 및 편집 (예: DALL·E, Stable Diffusion)
  • 코드 생성: 함수 작성, 버그 수정 보조
  • 오디오/음악 생성: 음성 합성, 배경 음악 생성

장점

창의성 보조, 반복 작업 자동화, 빠른 시제품 제작이 가능하며 다양한 산업에 적용되고 있습니다.

주의사항

잘못된 정보(허위), 편향(Bias), 저작권 문제, 악용 가능성 등 윤리적·법적 이슈를 고려해야 합니다.
